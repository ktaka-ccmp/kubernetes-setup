# kubernetes-setup

## Setup

### On Master

```
apt-get update ; apt-get install -y git ; git clone https://github.com/ktaka-ccmp/kubernetes-setup.git
cd kubernetes-setup
./master.sh
```

### On Node

```
apt-get update ; apt-get install -y git ; git clone https://github.com/ktaka-ccmp/kubernetes-setup.git
cd kubernetes-setup
./node.sh
```
